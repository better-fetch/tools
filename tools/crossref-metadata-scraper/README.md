# Crossref Metadata Scraper

Search Crossref Works or fetch one DOI through the public Crossref REST API and return normalized scholarly citation metadata with Better Fetch.

## Local validation

```bash
npm install
BETTER_FETCH_API_KEY=bf_... npm run test
```

## Example

```bash
BETTER_FETCH_API_KEY=bf_... bf-tool run --example deep-learning-doi
```
