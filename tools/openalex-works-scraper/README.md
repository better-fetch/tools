# OpenAlex Works Scraper

Search OpenAlex scholarly works or fetch one OpenAlex work by OpenAlex ID, DOI, or DOI URL through the public OpenAlex API.

## Examples

```bash
npm run validate
npm run test
npm run start
```

The tool is intentionally bounded to one OpenAlex API request per run. It returns public OpenAlex metadata only and does not crawl publisher pages or download full-text files.
