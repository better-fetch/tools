# ClinicalTrials.gov Study Scraper

Search ClinicalTrials.gov studies or fetch one exact NCT record through the public ClinicalTrials.gov v2 API.

## Local validation

```bash
npm install
BETTER_FETCH_API_KEY=bf_... npm run test
```

## Example

```bash
BETTER_FETCH_API_KEY=bf_... bf-tool run --example recruiting-diabetes
```
