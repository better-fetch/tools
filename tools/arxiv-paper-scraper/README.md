# arXiv Paper Scraper

Search arXiv or fetch a specific arXiv ID through the official arXiv Atom API and return normalized preprint metadata with Better Fetch.

Thank you to arXiv for use of its open access interoperability. This tool was not reviewed or approved by, nor does it necessarily express or reflect the policies or opinions of, arXiv.

## Local validation

```bash
npm install
BETTER_FETCH_API_KEY=bf_... npm run test
```

## Example

```bash
BETTER_FETCH_API_KEY=bf_... bf-tool run --example gpt4-paper
```
