# Apple App Store Scraper

Search Apple App Store software results or look up an app by Apple track ID using Apple's public iTunes Search API.

Version 0.1 returns app metadata including track ID, bundle ID, title, developer, genre, rating, rating count, price, currency, App Store URL, icon, version, release dates, and description. Review extraction, top charts, privacy labels, and full publisher catalogs are intentionally left for later validated slices.
