# Instagram Discovery Search

Find Google-indexed public Instagram posts, reels, and profiles by hashtag or keyword. `trending_reels` adds a query-free public-search recency rank, falling back from the last day to week to month when needed. It is not Instagram's authenticated Explore ranking. Results remain explicitly best-effort and source-linked; use the main Instagram scraper for deeper enrichment after discovery.

```bash
bf-tool run --example hashtag-posts
bf-tool test
```
