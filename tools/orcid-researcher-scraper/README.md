# ORCID Researcher Scraper

Search public ORCID registry records or fetch one public ORCID record through the official ORCID Public API.

## Examples

```bash
npm run validate
npm run test
npm run start
```

The tool is intentionally bounded to one official API request per run. It returns public registry data only and does not access member-only, limited-access, or private ORCID fields.
