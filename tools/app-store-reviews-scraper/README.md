# App Store Reviews Scraper

Extract public Apple App Store review rows from Apple's iTunes customer-review RSS JSON feed.

Version 0.1 reads one public review page for a numeric App Store ID or App Store URL and returns review IDs, titles, body text, star ratings, app versions, authors, author URLs, publication dates, vote counts, and review URLs. Multi-country fan-out, all-page pagination orchestration, app-name discovery, and authenticated App Store Connect review workflows are intentionally left for later validated slices.
